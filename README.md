# Portfolio

## Introduction
This is my portfolio project to showcase my capabilities as I work my way towards to becoming a full stack engineer. The aim of this project is to provide a brief history of my work experience as well as show case the projects and languages/frameworks I have worked with.

### Technologies
The languages used for this project consist of HTML, CSS, and JavaScript; written using Atom.

### Design
Included in this project is also a wireframe I came up to guide the overall design I was looking for. It mirrors the flat design aesthetic that is trending right now.
